觀察()是什麼意思
        1.(1+2)*3                ->優先/先做
        2. if(){}                ->公式/結構 裡面放判斷/比較/布林值(undefined/null/0 -> 會跑到false)
        3. function apple(){}    ->公式/結構 裡面放參數(參數可有可無)
        4. apple()               ->呼叫/執行 裡面放資料(如果有定義参數,就有可能會要有資料)
        5. switch(){}            ->公式/結構 裡面放變數
        6.while(){}              ->公式/結構 裡面放判斷/比較/布林值(undefined/null/0 -> 會跑到false)
        7.do{}while()            ->公式/結構 裡面放判斷/比較/布林值(undefined/null/0 -> 會跑到false)
        8.for(){}                ->公式/結構
        9.for(var x in object){} ->公式/結構
        10.function(){}          ->公式/結構
        11.()=>{}                ->公式/結構
